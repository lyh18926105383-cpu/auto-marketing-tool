# 汽修智能营销工具

面向汽修门店的网页端营销辅助工具。通过导入包含客户里程信息的 Excel 表格，自动计算车辆的日均行驶里程，预测下次保养时间，并生成定制化的微信营销话术。

## 功能特性

- Excel/CSV 客户数据导入
- 日均里程智能预测算法
- 客户状态自动分类（正常/临界/逾期）
- 一键生成营销话术
- 数据本地存储（IndexedDB）

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173

### 构建生产版本

```bash
npm run build
```

## 使用指南

### 1. 门店设置

点击右上角「设置」按钮，配置：
- 门店名称（用于营销话术）
- 门店电话（用于营销话术）

### 2. 导入客户数据

点击「导入Excel」按钮，上传包含以下字段的 Excel 文件：

| 字段 | 说明 |
|------|------|
| 姓名 | 客户姓名 |
| 手机 | 联系电话 |
| 车牌号 | 车牌号（唯一标识） |
| 车型 | 车辆型号 |
| 本次里程 | 当前里程（km） |
| 本次日期 | 进店日期 |

> 注意：重复导入相同车牌号时，旧数据会自动移入历史记录

### 3. 客户管理

- 左侧列表显示所有客户
- 支持按车牌号或姓名搜索
- 状态色块标识：绿色=正常，橙色=临界，红色=逾期

### 4. 生成营销话术

点击客户卡片，右侧显示：
- 预测数据（日均里程、预计到店日期）
- 营销话术预览
- 「一键复制」按钮

## 数据算法

### 日均里程计算
```
D = (当前日期 - 上次日期) / (当前里程 - 上次里程)
默认值：D = 40 km/天
```

### 下次保养里程
```
T = ceil(当前里程 / 5000) × 5000
（若已接近保养点，加 5000）
```

### 预测到店日期
```
P = 当前日期 + (T - 当前里程) / D
```
